# 🍽️ Zomato Restaurant Analysis

## 📌 Project Overview
This project focuses on Exploratory Data Analysis (EDA) of the Zomato Restaurant Dataset using Python. The objective is to analyze restaurant data, identify popular cuisines, and visualize insights through charts and graphs.

## 📂 Dataset
- Zomato Restaurant Dataset

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- WordCloud
- Google Colab

## 📊 Analysis Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Top 10 Popular Cuisines
- WordCloud Visualization
- Restaurant Data Insights

## 📈 Project Output
The project includes visualizations such as:
- Top 10 Popular Cuisines Bar Chart
- WordCloud of Restaurant Cuisines

## 🚀 How to Run
1. Open the notebook in Google Colab or Jupyter Notebook.
2. Install the required Python libraries.
3. Run all cells to generate the analysis and visualizations.

## 📌 Conclusion
This project demonstrates the use of Python for data cleaning, analysis, and visualization to extract meaningful insights from restaurant data.

# Loan Approval Prediction using Machine Learning

## InternSpark Data Science Internship - Task 2

## Project Overview

This project aims to predict whether a loan application will be approved based on applicant information using supervised machine learning techniques.

## Objective

Develop a machine learning model capable of predicting loan approval after performing data preprocessing, feature engineering, handling class imbalance, and evaluating multiple models.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- imbalanced-learn (SMOTE)

## Data Preprocessing

- Missing Value Handling
- Label Encoding
- Feature Scaling
- SMOTE for Class Balancing

## Machine Learning Models

- Logistic Regression
- Random Forest Classifier

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

## Results

| Model | Accuracy | Precision | Recall | F1 Score | ROC-AUC |
|------|---------:|----------:|--------:|---------:|---------:|
| Logistic Regression | 83.74% | 84.95% | 92.94% | 88.76% | 0.833 |
| Random Forest | 80.49% | 84.27% | 88.24% | 86.21% | 0.772 |

## Business Interpretation

- Good credit history significantly improves loan approval chances.
- Logistic Regression achieved the best overall performance.
- A threshold around **0.60** can be considered for deployment after business validation.
