# Loan Approval Prediction using Machine Learning

## InternSpark Data Science Internship - Task 2

## Project Overview

This project aims to predict whether a loan application will be approved based on applicant information using supervised machine learning techniques.

## Objective

Develop a machine learning model capable of predicting loan approval after performing data preprocessing, feature engineering, handling class imbalance, and evaluating multiple models.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- imbalanced-learn (SMOTE)

## Data Preprocessing

- Missing Value Handling
- Label Encoding
- Feature Scaling
- SMOTE for Class Balancing

## Machine Learning Models

- Logistic Regression
- Random Forest Classifier

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

## Results

| Model | Accuracy | Precision | Recall | F1 Score | ROC-AUC |
|------|---------:|----------:|--------:|---------:|---------:|
| Logistic Regression | 83.74% | 84.95% | 92.94% | 88.76% | 0.833 |
| Random Forest | 80.49% | 84.27% | 88.24% | 86.21% | 0.772 |

## Business Interpretation

- Good credit history significantly improves loan approval chances.
- Logistic Regression achieved the best overall performance.
- A threshold around **0.60** can be considered for deployment after business validation.

Task 3: Instagram Engagement Analysis
Project Overview

This project focuses on analyzing Instagram engagement data to identify user interaction patterns, content performance, hashtag usage, posting behavior, and follower growth signals. The analysis was performed using Python and data visualization techniques to generate meaningful insights.

Objective
Analyze likes and comments across Instagram posts.
Identify the most engaging posts.
Analyze hashtag usage.
Study different content types.
Determine the best posting time.
Analyze follower growth and activity.
Generate actionable insights from the dataset.

Key Insights
