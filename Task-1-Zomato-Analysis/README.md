# 🍽️ Zomato Restaurant Data Analysis

## 📌 Project Overview

This project performs an end-to-end Exploratory Data Analysis (EDA) on the Zomato Restaurant Dataset.

The objective is to understand customer preferences, restaurant trends, pricing patterns, cuisines, online ordering behaviour and factors affecting restaurant ratings.

This project has been developed as part of the InternSpark Data Science Internship.

---

## 🎯 Business Objective

The primary goal of this project is to generate meaningful business insights that can help restaurant owners and food delivery platforms improve customer experience and increase business growth.

---

## 📂 Dataset

Source:
Kaggle – Zomato Dataset

The dataset contains restaurant information including:

- Restaurant Name
- Location
- Cuisine
- Average Cost
- Online Order
- Table Booking
- Votes
- Rating
- Restaurant Type

## 💡 Key Insights

- Popular cuisines attract higher customer engagement.

- Restaurants offering online ordering generally receive more customer interactions.

- Premium restaurants usually have higher average ratings.

- Certain locations contain significantly more restaurants than others.

- Customer votes strongly influence restaurant popularity.

---

## 🚀 Business Recommendations

- Increase partnerships in high-performing locations.

- Promote restaurants with consistently high ratings.

- Encourage online ordering for better customer reach.

- Improve visibility of highly-rated restaurants.

- Personalize recommendations based on cuisine preference.
